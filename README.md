# Video Game Sales Forecasting
 This project aims to provide additional practice on common software engineering tasks, as well as identifying patterns that determine whether a game succeeds or not. This will allow the ability to spot potential big winners and plan advertising campaigns.
## Installation
To run the application locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/michav95/Integrated-Project-1
   ```
2. Navigate to the project directory:
   ```bash
   cd Integrated-Project-1
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
